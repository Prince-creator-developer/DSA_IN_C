Repository: DSA_IN_C
Description:

> Developed a comprehensive repository of standard Data Structures and Algorithms using C, emphasizing pointer arithmetic, memory allocation, and algorithmic optimization.

Contents & Topics Covered:
The repository contains C source codes (`.c`) and compiled executables (`.exe`) for various Data Structures and Algorithms, including:

* Arrays:
* `Arrays.c`, `array_mal_reall.c`

  
* Trees:
* Binary Search Trees (`BST.c`)
* B-Trees (`Btree.c`)
* Heaps (`heap.c`, `heap_min_max.c`, `heap_more.c`)


* Linked Lists:
* Singly Linked List (`linked_list.c`)
* Doubly Linked List (`doublyLL.c`, `doublyLLALL.c`, `app_DLL.c`)
* Circular Linked List (`circular_LL.c`)
* Polynomials using Linked List (`poly_with_LL.c`)


* Queues:
* Standard Queue Operations (`queue_ops.c`)
* Circular Queue (`circular_queue.c`)
* Queue with Linked List (`simplify_Q_with_LL.c`)


* Stacks & Expressions:
* Infix to Postfix Conversion (`infix2postfix.c`, `new_IN2POS.c`)
* Postfix Evaluation (`postfix_evaluation.c`)
* Palindrome check using Stack (`palindrome_stack.c`)


* Graphs:
* BFS and DFS (`bfs_dfs.c`, `bfs_dfs_II.c`, `graph_bfs_dfs.c`)


* **Hashing:** Linear Probing (`linear_probe.c`)

**Note:** The repository also includes a PDF file named `index_delayed_ui.pdf` and a TeX file `ok.tex`, which might contain further documentation or academic notes related to the code.
